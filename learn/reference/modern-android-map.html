Now in Android 学习速查地图

把项目拆成五个反复回看的坐标：Architecture、Modularization、Testing、UI、Performance。

Architecture

一条数据路径

ForYouScreen -> ForYouViewModel -> GetUserNewsResourcesUseCase -> NewsRepository + UserDataRepository -> Room/DataStore/Network。

Modularization

一个依赖方向

app 组装功能；feature:*:api 暴露导航入口；feature:*:impl 实现屏幕；core:* 提供共享能力。

Testing

一种测试判断

ViewModel 用 test repositories；data layer 用 fake DAO/network；Compose screen 用 Compose test；视觉回归用 Roborazzi；跨 Activity 流程用 instrumented tests。

UI

三层 UI 结构

Design system 定义基础组件和主题；core UI 放可复用业务组件；feature screen 组合 state、events 和 navigation。

Performance

两个性能证据

benchmarks 用 Macrobenchmark 测启动和关键路径；baseline-prof.txt 让 release 构建提前编译关键路径。
