# Teaching Notes

- 用户希望通过 Now in Android 学会现代 Android 应用的架构、模块化、测试、UI、性能。
- 教学语言以中文为主，关键 Android/Kotlin/Gradle/Compose/Flow 术语保留英文。
- 教学方式应围绕当前仓库真实代码展开，优先给可执行的阅读路径和练习。
- 用户在阅读 Compose 和 Kotlin Flow 源码时会被英文注释、状态管理、Modifier、布局约束、Flow 组合等基础概念拖慢。后续课程需要穿插“源码阅读前置知识”，先解释代码片段背后的 Compose/Kotlin 体系，再回到 Now in Android 架构。
